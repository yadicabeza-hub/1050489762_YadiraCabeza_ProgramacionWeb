Presentación del proyecto — Generar PowerPoint

Requisitos:
- Python 3.10+ (o la versión disponible)
- pip
- Paquete: python-pptx

Instalación:
```bash
pip install python-pptx
```

Generar la presentación:
```bash
cd presentation
python build_presentation.py
```

Salida:
- `presentacion_proyecto_pw.pptx` en la carpeta `presentation`.

Notas:
- El script usa un contenido básico embebido. Puedes editar `presentation/slides_content.md` para el contenido y mejorar el script para parsear el Markdown.
- Si quieres que yo genere el `.pptx` aquí, puedo intentar ejecutarlo si confirmas instalar dependencias y permites la ejecución.
